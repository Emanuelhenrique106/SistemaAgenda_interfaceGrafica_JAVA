# SistemaAgenda_interfaceGrafica_JAVA

# Agenda de Contatos Simples

Este é um projeto básico de sistema de agenda de contatos desenvolvido em Java com interface gráfica utilizando Swing. O sistema permite cadastrar e visualizar informações de contatos de forma simples e intuitiva, servindo como um ótimo ponto de partida para quem deseja aprender sobre interfaces gráficas em Java e manipulação de dados básicos.

# Funcionalidades

Cadastro de contatos com informações básicas (nome, sobrenome, telefone)
Interface gráfica simples e direta, organizada em GridLayout
Estrutura modular que facilita futuras melhorias e expansão do sistema.

# Tecnologias utilizadas

Java JDK
Swing (GUI)
IDE: Eclipse.

# Objetivo

Este projeto tem caráter educacional, sendo ideal para iniciantes que querem praticar:
Criação de interfaces gráficas com Java
Estruturação básica de sistemas orientados a objetos
Manipulação de dados de forma simples.
